# Tablet Mounted on the Wall for HA
Tablet mounted on the wall in the Living room used for controlling HA

## Needed Parts
- Lenovo M10 FHD 2nd gen Tablet
- USB-C connector
- Using 2 of the non-used thermostat internal wires
